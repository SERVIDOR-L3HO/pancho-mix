# Replit Agent Skills Repository

## Overview
This repository is a collection of skills and artifacts used by Replit AI agents. It contains:

- **`.local/skills/`** — Documentation and tooling instructions for specialized agent tasks (testing, Stripe integration, canvas manipulation, database management, etc.)
- **`.local/skills/artifacts/`** — Project scaffold templates the agent can deploy (React+Vite web apps, Expo mobile apps, Mastra automation workflows, mockup sandboxes)

There is no runnable application in this repository — it is a configuration and reference library for Replit's agent system.

## User Preferences
- This project uses pnpm as the package manager where applicable.
- Primary language: TypeScript.
